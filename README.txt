To run the project:
1) Create a directory named project_datasets in the base directory (the one containing this file)
2) Copy the .mat data files for the EEG channels into that file so that there are 9 .mat files
3) Open a terminal and cd to the /neural_net/utilities/ directory
4) Run $ python new_preprocess.py
5) cd to the base directory (the one containing this file)
6) Run $ python main.py

To run the project:
1) Install Keras and Tensorflow
2) Create a directory named project_datasets in the base directory (the one containing this file)
3) Copy the .mat data files for the EEG channels into that file so that there are 9 .mat files
4) Open a terminal and cd to the /neural_net/utilities/ directory
5) Run $ python new_preprocess.py
6) cd to the base directory (the one containing this file)
7) Run $ python main.py
